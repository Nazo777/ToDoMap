# ToDoMap
Heroku: https://todomap777.herokuapp.com
# 概要
行きたい場所をマップにメモするアプリ
# 機能一覧
- マップ表示
- マーカー表示
- マップ検索機能
- 吹き出し表示
- リスト表示
- リスト投稿機能
- リスト編集機能
- ユーザーログイン/ログアウト機能
- ユーザー簡単ログイン機能
- ユーザー新規登録機能
# 開発環境
- Ruby
- RubyonRails
- HTML/CSS
- Javascript
- GoogleMapsAPI
- Git/Github
- HEROKU
- mysql
- ClearDB
- VSCode
